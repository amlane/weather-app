# Weather App 5000

A front-end web application built using:

- React JS
- Axios
- Open Weather API
- Reverse Geocoding API built by Texas A&M
- Geolocation API in JavaScript
- Moment.js
- React loader spinner

Deployed using Netlify @ https://weatherapp5000.netlify.app/
